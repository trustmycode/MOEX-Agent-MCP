## Описание

Собрать и структурировать корпус из 20–40 документов (методики МОEX/ЦБ, гайды по риск-менеджменту, шаблоны отчётов) в каталоге `rag_corpus/` с краткими описаниями для каждого документа.

## Критерии приемки

- В репозитории создан каталог `rag_corpus/` (или эквивалент) со списком документов.
- Для каждого документа зафиксированы:
  - источник (МОEX, ЦБ, внутренний документ и т.п.);
  - краткое описание содержания;
  - язык и дата/версия.
- Есть отдельный список/таблица, пригодный для документирования в README/архитектуре.

## Определение готовности

- Корпус достаточно полон, чтобы покрывать основные понятия волатильности, доходности, риск-менеджмента и отчётности.
