# TASK-2025-107: Демонстрационные сценарии и сторителлинг для жюри (портфельный риск + CFO)

## Статус

- Статус: planned
- Приоритет: P0
- Компонент: бизнес-агент / демо-скрипты / документация
- Связанные сценарии: portfolio_risk (7), cfo_liquidity_report (9), issuer_peers_compare (5) — опционально

## Контекст

Для хакатона важно не только наличие функционала, но и качественный сторителлинг: понятные демо-сценарии, которые стабильно воспроизводятся и показывают value для риск-менеджера и CFO на реальных данных MOEX.

## Цель

Подготовить полный набор демо-скриптов, примеров запросов и документации, которые проводят жюри через «геройскую связку» `portfolio_risk` + `cfo_liquidity_report` и, при наличии времени, `issuer_peers_compare`.

## Объём работ

### In scope

- Описание персонажей:
  - риск-менеджер (фокус на концентрациях, стрессах, Var_light),
  - CFO (фокус на ликвидности, валютных рисках, рефинансировании).
- Скрипты демо:
  - пошаговое описание, что вводит пользователь, что отвечает агент,
  - тексты для презентации/питча.
- Технические скрипты:
  - команды/запросы для запуска сценариев 7 и 9 из консоли/интерфейса Evolution,
  - подготовка фиксированного тестового портфеля (JSON/CSV).
- Документация:
  - раздел «How to demo» в README/доках,
  - ссылки на примеры отчётов (из TASK-2025-105).

### Out of scope

- Полноценная презентация в слайдах (может быть сделана отдельно).
- Видео-демо.

## Acceptance Criteria

- [ ] Существуют текстовые демо-скрипты для сценариев 7 и 9 (и, по возможности, 5).
- [ ] Подготовлен тестовый портфель и примеры команд/запросов для воспроизведения демо.
- [ ] В README/документации есть раздел «How to demo», опирающийся на эти скрипты.
- [ ] В ходе внутреннего прогона демо сценарии 7 и 9 отрабатывают стабильно и без неожиданностей.
