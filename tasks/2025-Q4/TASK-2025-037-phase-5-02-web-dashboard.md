## Описание

Реализовать одностраничный веб-дашборд (SPA на Svelte/React/Next или другом стеке) для отправки запросов агенту, отображения текстового ответа, таблиц и базовых графиков.

## Критерии приемки

- UI предоставляет:
  - поле ввода запроса;
  - область для отображения `output.text`;
  - таблицы из `output.tables[]`.
- Для сценария «Сравни SBER и GAZP за год» UI показывает:
  - текстовое резюме;
  - таблицу с метриками по обоим тикерам;
  - как минимум один график (временной ряд цены/доходности или бар-чарт по top-5 весам).

## Определение готовности

- Веб-дашборд стабильно работает в связке с агентом и пригоден для живого демо.
