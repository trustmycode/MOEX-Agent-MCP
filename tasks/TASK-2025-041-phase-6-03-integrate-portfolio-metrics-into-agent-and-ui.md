## Описание

Интегрировать `get_portfolio_metrics` в Planner и ResponseFormatter агента, добавить соответствующий режим/вкладку в UI для анализа портфеля и визуализации распределения по бумагам/сектору.

## Критерии приемки

- Planner распознаёт запросы вида «вот портфель, оцени риск/доходность» и добавляет шаг `get_portfolio_metrics` в план.
- `ResponseFormatter`:
  - формирует отдельную таблицу по портфелю;
  - поясняет «красные флаги» на понятном бизнес-языке;
  - при наличии beta объясняет, как соотносится риск портфеля с рынком.
- UI предоставляет режим «Анализ портфеля» с:
  - диаграммой распределения по бумагам/сектору;
  - (по возможности) визуализацией вклада бумаг в риск.

## Определение готовности

- Сценарий «Вот мой портфель, оцени риск и предложи рекомендации» работает end-to-end и удобно демонстрируется в UI.
