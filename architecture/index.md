# Обзор архитектуры moex-market-analyst-agent

Этот раздел описывает фактическую архитектуру системы на текущий момент с опорой на существующие документы в `docs/` и план архитектора.

## Структура архитектурных документов

- `architecture/agent/ARCH-agent-moex-market-analyst-v1.md` — компонент AI-агента (A2A), который обрабатывает запросы пользователей и оркестрирует вызовы MCP.
- `architecture/mcp/ARCH-mcp-moex-iss-v1.md` — сервисный компонент MCP-сервера `moex-iss-mcp`, предоставляющий бизнес‑ориентированные инструменты поверх MOEX ISS API.

## Текущее состояние

- Исходный код ещё не добавлен в репозиторий, поэтому архитектура отражает целевой дизайн, согласованный в документах `docs/ARCHITECTURE.md`, `docs/REQUIREMENTS_moex-market-analyst-agent.md` и `docs/SPEC_moex-iss-mcp.md`.
- Все новые архитектурные документы имеют статус `current` и будут актуализироваться по мере появления реализации.
